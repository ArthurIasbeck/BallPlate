\doxysubsection{Descrição detalhada}
Coleta de dados da tela touch para averiguação de ruídos. 

\begin{DoxyAuthor}{Autor}
Arthur Iasbeck (\href{mailto:arthuriasbeck@gmail.com}{\texttt{ arthuriasbeck@gmail.\+com}})
\end{DoxyAuthor}
Neste arquivo são implementadas funções para coleta de valores da tela touch. Os valores são coletados e enviados ao PC juntamente com tempo de amostragem. O objetivo desta coleta é determinar a variância do ruído de medida para projeto do Filtro de Kalman. Vale ressaltar que o experimento realizado dura 20 segundos e seu fim é ditado pelo Arduino. \begin{DoxyVersion}{Versão}
